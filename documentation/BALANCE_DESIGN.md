# PETS GO Lite - Balance Design Document

## Контекст

Полная документация по системе баланса PETS GO Lite. Основана на принципах оригинальной PETS GO (Roblox, BIG Games): индивидуальные шансы питомцев ("1 из X"), яйца как фильтр минимальной редкости, мультипликативные бафы удачи.

Адаптирована под веб-платформу: ручные клики, короткие сессии, монетизация через rewarded video ads.

---

## 1. Система шансов питомцев

### 1.1 Принцип "1 из X"

Каждый питомец имеет **индивидуальный** шанс выпадения, записанный как **"1 из X"** (1kX). Чем больше X, тем реже пет. Редкость (Common, Rare...) — это категория, определяемая диапазоном шанса.

## 2. Алгоритм ролла

### 2.1 Как определяется результат

Система использует **последовательную проверку от редчайшего к обычному** — как в оригинальной PETS GO. Никаких весов, нормализации, пулов. Чистая математика: каждый пет проверяется независимо.

**Точный алгоритм:**

```
function roll(eligiblePets, luckMultiplier):
    // Сортируем: от самого редкого (большой X) к обычному (малый X)
    sorted = eligiblePets.sortByChanceDesc()

    for each pet in sorted:
        // Шанс пройти проверку = luck / X
        checkChance = min(1.0, luckMultiplier / pet.X)
        if random() < checkChance:
            return pet     // ← ПЕРВЫЙ прошедший = результат. СТОП.

    // Если НИКТО не прошёл — возвращаем самого обычного
    return sorted[last]
```

**Ключевые принципы:**
- **Нет весов.** Каждый пет = отдельный бросок кубика
- **Нет нормализации.** "1 из 1000" означает ровно 1/1000
- **Порядок важен.** Редкие проверяются ПЕРВЫМИ
- **Самый обычный пет = fallback.** До него дело доходит, только если все остальные не прошли
- Множитель удачи просто **делит X**: пет 1/1000 с удачей x5 = проверка 5/1000 = 1/200

### 2.2 Почему обычные петы теряют долю

Это происходит **естественно**, без всяких cap'ов или хаков. Механизм прост:

```
Удача x5, проверяем 6 петов:

  Gecko  (1/1000): проверка = 5/1000 = 0.5%  → скорее нет...
  Fox    (1/30):   проверка = 5/30  = 16.7%  → может быть!
  Squirrel(1/10):  проверка = 5/10  = 50%    → часто проходит!
  Chick  (1/5):    проверка = 5/5   = 100%   → ГАРАНТИЯ если дошли!
  Dog    (1/3):    НЕ ПРОВЕРЯЕТСЯ — Chick уже 100%, до Dog не дойдёт
  Cat    (1/2):    НЕ ПРОВЕРЯЕТСЯ — до Cat не дойдёт
```

При удаче x5 Chick проходит проверку со 100% вероятностью. Раз Chick гарантирован, Dog и Cat **физически не могут выпасть** — до них очередь не доходит!

### 2.4 Как это влияет на игровой опыт

Эта механика создаёт **естественное "окно" петов** для каждого уровня удачи:

| Удача | "Окно" доступных петов | Что происходит |
|-------|----------------------|---------------|
| x1 | Все (Cat доминирует) | Новичок, много обычных |
| x2 | Cat падает, Dog/Chick растут | Уже интереснее |
| x5 | Cat/Dog исчезают, Squirrel/Chick доминируют | Uncommon реальны |
| x10 | Squirrel+ блокирует, Fox становится частым | Rare начинают появляться |
| x30 | Fox гарантирован, Gecko = 3% | Только Uncommon+ |

Чем выше удача, тем **выше "пол"** — обычные петы просто перестают выпадать, и ты автоматически играешь среди более редких.

## 3. Система яиц

### 3.1 Принцип работы

Яйца **автоматически** активируются при достижении определённого уровня игрока. Каждое яйцо задаёт **минимальный порог**: из пула роллов убираются все питомцы обычнее указанного порога.

### 3.2 Таблица яиц

| Яйцо | Уровни | Фильтр | Убирает | Остаётся петов |
|-------|--------|--------|---------|---------------|
| **Basic Egg** | 1–5 | Нет (все 1/2+) | Никого | 30 |
| **Golden Egg** | 6–12 | 1/3+ | Cat (1/2) | 29 |
| **Crystal Egg** | 13–22 | 1/6+ | Cat, Dog, Bunny, Chick | 26 |
| **Fire Egg** | 23–35 | 1/12+ | +Chicken, Lamb, Piggy, Squirrel | 22 |
| **Cosmic Egg** | 36+ | 1/20+ | +Cow, Froggy, Turkey | 19 |

### 3.3 Детализация по яйцам

**Basic Egg (Level 1–5):** Все 30 питомцев доступны. Новичок видит в основном Cat, Dog, Bunny.

**Golden Egg (Level 6–12):** Убирает Cat (1/2). Dog (1/3) становится самым частым. Начинают чаще попадаться Uncommon.

**Crystal Egg (Level 13–22):** Убирает всех обычнее 1/6. Chicken (1/6) — самый частый. Uncommon появляются регулярно.

**Fire Egg (Level 23–35):** Убирает всех обычнее 1/12. Cow (1/12) — самый частый. Rare появляются ощутимо.

**Cosmic Egg (Level 36+):** Убирает всех обычнее 1/20. Goldfish (1/20) — самый частый из Common. Большая часть роллов — Uncommon и выше. Endgame-яйцо.

### 3.4 Как яйцо влияет на вероятности

Когда яйцо отсекает обычных петов, вероятности оставшихся **перенормируются**. Это автоматически повышает шанс редких.

**Пример: Cosmic Egg (1/20+) без бафов:**

Сумма весов оставшихся 19 петов значительно меньше, чем всех 30. Доля Legendary растёт:

```
С Basic Egg:  Blobfish (1/50000) ≈ 0.001% от пула
С Cosmic Egg: Blobfish (1/50000) ≈ 0.006% от пула (в ~6 раз чаще!)
```

## 4. Виды роллов и бафы удачи

### 4.1 Обычный ролл (Normal Roll)

**Множитель удачи:** x1 (без бонусов)
**Стоимость:** бесплатно, всегда доступен

**Механика (пошагово):**

1. Игрок нажимает кнопку **Roll** → кнопка блокируется (неактивный стиль)
2. Экран **затемняется** (тёмный оверлей)
3. Появляется **яйцо** текущего тира (по уровню игрока)
4. Яйцо **трясётся** (анимация shake)
5. Яйцо **разбивается** (смена спрайта на разбитое яйцо того же тира)
6. Одновременно: питомец появляется с **zoom-in** анимацией
7. Отображается: **картинка питомца**, **название**, **редкость**, маркер **"New"** (если первый раз)
8. Если питомец новый → **маркер нотификации** на кнопке Коллекция. Маркер **пульсирует** (увеличение на 15%, два коротких раза) каждые 3 секунды. Исчезает при открытии коллекции.
9. Для редких питомцев (Rare+) → **эффект свечения** на слое ПОД питомцем (вращающийся glow, показывающий редкость)
10. Питомец отображается **2 секунды**, затем исчезает
11. **Затемнение** снимается
12. Кнопка Roll снова **активна**
13. Кнопка Roll **пульсирует** (увеличение на 10%, два раза) каждые 15 секунд для привлечения внимания

**Визуальные эффекты по редкости:**

| Редкость | Эффект при появлении |
|----------|---------------------|
| Common | Нет дополнительных эффектов |
| Uncommon | Нет дополнительных эффектов |
| Rare | Вращающееся свечение (голубое) под питомцем |
| Epic | Вращающееся свечение (фиолетовое) + частицы |
| Legendary | Вращающееся свечение (золотое) + частицы + flash экрана |

---

### 4.2 Lucky Roll (Удачный ролл) — за рекламу

**Множитель удачи:** x2
**Получение:** за просмотр рекламы
**Количество:** 3 ролла за одну рекламу (предварительно)
**Доступность:** всегда, без кулдауна
**Цвет плашки:** Зелёный (как лак)

**Механика:**

1. Кнопка **"Lucky Roll"** отображается постоянно в зоне бафов (справа от Roll)
2. Рядом с ней — **иконка видео/плёнки** (Poki requirement)
3. Игрок нажимает кнопку
4. Запускается **rewarded video ad** (через PlatformSDK)
5. **После успешного завершения рекламы** (callback `success: true`):
   - Игроку начисляются **3 Lucky Roll**
   - Над кнопкой Roll появляется **плашка "Lucky x3"** зелёного цвета
6. Если реклама прервана / закрыта досрочно → **бонус НЕ начисляется**
7. Игрок нажимает Roll → используется **1 Lucky Roll** (счётчик: 3→2→1→0)
8. Каждый Lucky Roll применяет **x2 к множителю удачи**
9. При достижении 0 → плашка исчезает
10. Кнопка "Lucky Roll" снова доступна для нового просмотра

**Важно:** Lucky Roll можно **накапливать**. Если у игрока осталось 2 Lucky Roll и он посмотрел ещё одну рекламу, будет 2 + 3 = 5 Lucky Roll.

---

### 4.3 Super Roll (Супер ролл) — за рекламу с кулдауном

**Множитель удачи:** x3
**Получение:** за просмотр рекламы
**Количество:** 1 ролл за одну рекламу (предварительно)
**Кулдаун предложения:** 30 секунд (предварительно)
**Цвет плашки:** Оранжевый

**Механика:**

1. С анимацией **выезжания справа** появляется кнопка Super Roll
2. Рядом — **иконка видео/плёнки**
3. Игрок нажимает кнопку
4. Запускается **rewarded video ad**
5. **После успешного завершения рекламы:**
   - Игроку начисляется **1 Super Roll**
   - Над кнопкой Roll появляется **плашка "Super x1"** оранжевого цвета
6. Кнопка Super Roll **уезжает вправо** (анимация)
7. Запускается **внутренний таймер** (30 секунд)
8. Игрок нажимает Roll → используется **1 Super Roll**
9. Super Roll применяет **x3 к множителю удачи**
10. По истечении таймера → кнопка Super Roll снова **выезжает справа**
11. Цикл повторяется с пункта 1

**Super Roll накапливается.** Если игрок не использовал Super Roll, а таймер истёк и он посмотрел новую рекламу — будет 2 Super Roll.

---

### 4.4 Epic Roll (Эпик ролл) — за время в игре

**Множитель удачи:** x5
**Получение:** бесплатно, за время пребывания в игре
**Количество:** 1 ролл за каждый период ожидания
**Таймер обновления:** 30 секунд (предварительно)
**Цвет плашки:** Красный/Огненный

**Механика:**

1. С анимацией **выезжания справа** появляется кнопка Epic Roll (неактивная, с таймером)
2. Отображается **таймер обратного отсчёта** (30 сек) на кнопке
3. Когда таймер достигает 0:
   - Кнопка **меняется на активную** (подсветка, glow-эффект)
   - Опционально: **звук уведомления** / **пульсация** кнопки
4. Игрок нажимает кнопку → получает **1 Epic Roll**
5. Над кнопкой Roll появляется **плашка "Epic x1"** красного цвета
6. Таймер **сбрасывается** и начинает отсчёт заново
7. Игрок нажимает Roll → используется **1 Epic Roll**
8. Epic Roll применяет **x5 к множителю удачи**

**Epic Roll накапливается.** Если игрок не использовал Epic Roll, а таймер снова истёк — будет 2 Epic Roll, потом 3, и т.д. Максимум накопления — 10 (предварительно).

**Особенность:** Epic Roll — единственный баф, который **не требует рекламы**. Это награда за время в игре, мотивирующая длинные сессии.

### 4.5 Стакинг бафов (перемножение)

Все бафы **перемножаются** между собой. Игрок может одновременно иметь активными Lucky Roll, Super Roll и Epic Roll.

**Формула итогового множителя:**

```
luckMultiplier = luckyRollMultiplier * superRollMultiplier * epicRollMultiplier

Где:
  luckyRollMultiplier = 2 (если есть Lucky Roll) или 1 (если нет)
  superRollMultiplier = 3 (если есть Super Roll) или 1 (если нет)
  epicRollMultiplier  = 5 (если есть Epic Roll) или 1 (если нет)
```

**Таблица комбинаций:**

| Активные бафы | Множитель | Эффект на Blobfish (1/50,000) |
|---------------|-----------|-------------------------------|
| Нет бафов | x1 | 1 из 50,000 (0.002%) |
| Lucky Roll | x2 | 1 из 25,000 (0.004%) |
| Super Roll | x3 | 1 из 16,666 (0.006%) |
| Epic Roll | x5 | 1 из 10,000 (0.01%) |
| Lucky + Super | x6 | 1 из 8,333 (0.012%) |
| Lucky + Epic | x10 | 1 из 5,000 (0.02%) |
| Super + Epic | x15 | 1 из 3,333 (0.03%) |
| **Lucky + Super + Epic** | **x30** | **1 из 1,666 (0.06%)** |

### 4.6 Визуальное отображение бафов

Плашки бафов отображаются **слева направо** над кнопкой Roll:

```
┌──────────┐  ┌──────────┐  ┌──────────┐
│ Lucky x3 │  │ Super x1 │  │ Epic x2  │
│  (зелён) │  │ (оранж)  │  │  (красн) │
└──────────┘  └──────────┘  └──────────┘
              ┌──────────────────────────┐
              │         ROLL!            │
              └──────────────────────────┘
```

При использовании ролла **один баф каждого типа списывается за ролл** (Lucky: 3→2, Super: 1→0, Epic: 2→1). Плашка с 0 исчезает.

**Приоритет списания:** списываются **все активные бафы одновременно**. Один ролл потребляет по 1 от каждого типа. Это мотивирует копить бафы и использовать их стеком.

## 5. Начисление рекламных бонусов

### 5.1 Когда начисляется бонус

Бонус начисляется **строго после успешного завершения рекламы**. Это стандартная практика для всех рекламных платформ.

**Порядок:**

```
1. Игрок нажимает кнопку (Lucky Roll / Super Roll)
2. Игра ставится на паузу, звук выключается
3. Вызывается PlatformSDK.showRewardedBreak()
4. Реклама воспроизводится
5. SDK возвращает callback:
   - success: true  → игрок досмотрел → НАЧИСЛИТЬ бонус
   - success: false → игрок закрыл → НЕ начислять
6. Игра возобновляется, звук включается
```

### 5.2 Реализация для разных платформ

```typescript
// Poki
PokiSDK.rewardedBreak(() => { /* mute */ })
  .then((success: boolean) => {
    if (success) grantReward();  // ← только здесь
    resumeGame();
  });

// CrazyGames
CrazyGames.SDK.ad.requestAd('rewarded', {
  adStarted: () => { /* mute */ },
  adFinished: () => { grantReward(); resumeGame(); },  // ← успех
  adError: () => { resumeGame(); },                     // ← неудача
});
```

### 5.3 Защита от злоупотреблений

- Бонус начисляется **только после** callback'а от SDK, не до
- Если SDK недоступен (ad blocker) → кнопки рекламных бафов **скрываются**
- Epic Roll (за время) работает **всегда**, независимо от рекламы
- Первая реклама — **не раньше 2 минут** от начала сессии (Poki policy)

## 6. Яйца + Бафы = Полная формула

### 6.1 Итоговая формула ролла

```
1. Определить тир яйца по уровню игрока
2. Отфильтровать петов: убрать всех обычнее порога яйца
3. Отсортировать оставшихся от РЕДЧАЙШЕГО к ОБЫЧНОМУ
4. Определить множитель удачи = Lucky * Super * Epic (каждый = 1 если неактивен)
5. Для каждого пета (от редкого к обычному):
     checkChance = min(1.0, luckMultiplier / pet.X)
     if random() < checkChance → ВЕРНУТЬ этого пета, СТОП
6. Если никто не прошёл → вернуть самого обычного из пула
7. Списать по 1 от каждого активного бафа
```

### 6.2 Примеры реальных ситуаций

**Новичок (Level 1, Basic Egg, без бафов, x1):**

```
Доступные петы: все 30
Проверка идёт: Blobfish(1/50K) → ... → Gecko(1/1K) → ... → Fox(1/30) → ... → Cat(fallback)
Все редкие проверки провалены → Cat ~46%, Dog ~23%, Bunny ~14%
Legendary шанс: ~0.002% (1 из 50,000)
Ощущение: "получаю котиков и собачек"
```

**Игрок Level 8 (Golden Egg, Lucky Roll x2):**

```
Доступные петы: 29 (Cat убран яйцом)
Множитель: x2
Dog(fallback) ~15%, Bunny ~30%, Chick ~25%
Uncommon проверки: Fox 2/30=6.7% → реально появляются!
Ощущение: "начинаю видеть зелёных (Uncommon) петов!"
```

**Игрок Level 20 (Crystal Egg, Super + Epic = x15):**

```
Доступные петы: 26 (Cat, Dog, Bunny, Chick убраны)
Множитель: x15
Chicken(1/6): 15/6 > 1 → гарантирован если дойдёт, но Fox(1/30): 15/30=50% перехватит раньше
Fox и более частые Uncommon доминируют (~60%+ суммарно)
Rare Horse(1/200): 15/200 = 7.5% проверка → реальный шанс!
Ощущение: "Uncommon — мой хлеб, иногда ловлю Rare!"
```

**Ветеран Level 40 (Cosmic Egg, все бафы x30):**

```
Доступные петы: 19 (только 1/20+)
Множитель: x30
Goldfish(1/20, fallback): до него дело почти не доходит
Fox(1/30): 30/30=100% → гарантирован, блокирует всё обычнее
Blue Fish(1/100): 30/100=30% → частый
Gecko(1/1000): 30/1000=3% → реальный
Shark(1/2500): 30/2500=1.2% → возможный
Blobfish(1/50000): 30/50000=0.06% → ~6% за 100 роллов
Ощущение: "гоняюсь за Epic, Legendary — реальная мечта"
```

## 7. Прогрессия и XP

### 7.1 XP за ролл

| Событие | XP (% от уровня) |
|---------|-------------------|
| Новый Common пет | 25% |
| Новый Uncommon пет | 25% |
| Новый Rare пет | 25% |
| Новый Epic пет | 25% |
| Новый Legendary пет | 25% |
| Дупликат Common | 1% |
| Дупликат Uncommon | 2% |
| Дупликат Rare | 3% |
| Дупликат Epic | 5% |
| Дупликат Legendary | 10% |

### 7.2 Кривая XP

```
XP для уровня = 100 * (1.15 ^ (level - 1))

Level 1:  100 XP
Level 5:  175 XP
Level 10: 404 XP
Level 15: 814 XP
Level 20: 1,637 XP
Level 30: 6,621 XP
Level 40: 26,787 XP
Level 50: 108,366 XP
```

### 7.3 Темп прогрессии

На ранних уровнях (1–5) каждый новый пет = 25% уровня → 4 новых пета = level up. С учётом дупликатов — примерно **8–12 роллов на уровень**.

На поздних уровнях (30+) нужно ~6,600 XP, дупликат Common даёт ~66 XP → **100 роллов на уровень** (в основном через дупликаты).

## 8. Таймеры и экономика сессии

### 8.1 Типичная сессия (10 минут)

```
Время     Событие
0:00      Начало. Basic Egg. Нет бафов.
0:00      Кнопка Lucky Roll доступна (реклама)
0:30      Первый Epic Roll готов (таймер 30 сек)
1:00      Второй Epic Roll готов
1:30      Третий Epic Roll + Super Roll предложение появляется
2:00      Реклама разрешена (Poki 2-min policy)
...
10:00     Конец сессии

За 10 мин при ручных кликах (~1 ролл / 3 сек):
  ~200 роллов максимум (с учётом анимаций — реально ~100-120)

За 10 мин Epic Roll:
  ~20 штук (каждые 30 сек)

За 10 мин Super Roll (если каждые 30 сек реклама):
  3-5 штук (ограничено терпением игрока к рекламе)

За 10 мин Lucky Roll:
  1-3 просмотра = 3-9 Lucky Roll
```

### 8.2 Рекомендуемые таймеры

| Параметр | Значение | Обоснование |
|----------|----------|-------------|
| Epic Roll таймер | 30 сек | Достаточно часто для ощущения "бесплатной награды" |
| Epic Roll макс. накопление | 10 | Не слишком много, мотивирует использовать |
| Super Roll кулдаун предложения | 30 сек | Не надоедает, но напоминает о себе |
| Lucky Roll кулдаун | Нет | Всегда доступен — основная монетизация |
| Lucky Roll за просмотр | 3 шт | Ощутимо, но быстро кончается |
| Super Roll за просмотр | 1 шт | Ценнее, поэтому меньше |
| Длительность анимации ролла | ~3 сек | Яйцо + появление + показ |
| Показ результата | 2 сек | Достаточно для осознания |
| Пульсация кнопки Roll | каждые 15 сек | Ненавязчивое напоминание |
| Пульсация нотификации | каждые 3 сек | Более активное привлечение |

### 10.2 Алгоритм ролла (для `src/systems/RNGSystem.ts`)

```typescript
function roll(eligiblePets: PetDef[], luckMultiplier: number): PetDef {
    // Сортируем от редчайшего (большой X) к обычному (малый X)
    const sorted = [...eligiblePets].sort(
        (a, b) => PET_CHANCES[b.id] - PET_CHANCES[a.id]
    );

    // Последовательная проверка
    for (const pet of sorted) {
        const x = PET_CHANCES[pet.id];
        const checkChance = Math.min(1.0, luckMultiplier / x);
        if (this.rng.next() < checkChance) {
            return pet;  // Первый прошедший = результат
        }
    }

    // Fallback: самый обычный пет из пула
    return sorted[sorted.length - 1];
}
